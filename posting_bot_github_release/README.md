# @postingdbot — Бот для постинга в Telegram-каналы

Этот бот позволяет администраторам каналов создавать посты с медиафайлами, URL-кнопками, динамическими реакциями-опросами и планировать публикации на нужное время.

## 🚀 Основные возможности
- **Интерактивное добавление каналов**: Использование современного интерфейса Telegram для выбора каналов и проверки прав бота.
- **Гибкий конструктор постов**: Поддержка текста (с разметкой HTML), изображений, видео, документов, аудио, стикеров и гифок.
- **Интерактивные URL-кнопки**: Создание кнопок-ссылок с возможностью размещения нескольких кнопок в ряд.
- **Кнопки реакций с подсчетом голосов**: Добавление реакций-эмоджи под пост. Бот отслеживает голоса читателей в базе данных для предотвращения накруток.
- **Отложенная публикация**: Встроенный планировщик задач на базе `APScheduler`. Вы можете запланировать пост на любую дату и время по Московскому времени (UTC+3).
- **Разделение прав**: Пользователи видят и управляют только своими каналами.

## 🛠️ Установка и запуск

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python версии 3.9 или выше.

Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

### 2. Настройка конфигурации
1. Переименуйте файл `.env.template` в `.env` (или отредактируйте созданный по умолчанию файл `.env`).
2. Откройте `.env` и вставьте ваш токен бота от [@BotFather](https://t.me/BotFather):
   ```env
   BOT_TOKEN=1234567890:ABCdefGhIJKlmNoPQRsTUVwxyZ
   ```

### 3. Запуск бота
Запустите бота следующей командой:
```bash
python main.py
```

## 📖 Как пользоваться ботом

### Добавление канала:
1. Откройте бота и нажмите `/start`.
2. Нажмите кнопку **«📢 Мои каналы»**, затем **«➕ Добавить новый канал»**.
3. Выберите ваш канал. Бот автоматически запросит необходимые права администратора и проверит их корректность.

### Создание поста:
1. Нажмите **«✍️ Создать пост»** и выберите канал.
2. Отправьте контент (текст, картинку, видео и т.д.).
3. В меню предпросмотра вы можете:
   - Добавить URL-кнопки в формате: `Текст кнопки - ссылка` (для разделения кнопок в строке используйте `|`).
   - Добавить кнопки реакций (например: `👍 👎 🔥`).
   - Отредактировать текст или заменить медиафайл.
4. Выберите **«🚀 Опубликовать сейчас»** или **«📅 Отложить»** (в формате `ДД.ММ.ГГГГ ЧЧ:ММ` по МСК).
